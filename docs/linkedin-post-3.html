Installation Guide No Store Needed
1

Get the Code

Clone the repository or download the ZIP from GitHub and extract it anywhere.

git clone https://github.com/Ns81000/imdb-list-injector.git
2

Open Extensions

Open your Chrome browser, type this shortcut into the address bar, and hit Enter:

chrome://extensions
3

Developer Mode

Toggle the Developer mode switch in the top-right corner of the Extensions page.

4

Load Unpacked

Click Load unpacked in the top-left, select the extracted folder, and start exporting!

Unlock Immersive Mode optional

To project lists as backdrop-driven trailer slideshows, save a free TMDB key in settings:

  1. Get a free API key from TMDB Settings → API.
  2. Paste in Settings (gear icon) → TMDB API Key.
  3. Choose a local passphrase to encrypt and save.
